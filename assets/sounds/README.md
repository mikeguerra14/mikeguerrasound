# UI Sounds

This directory contains the UI sound effects for the website.

## Required Sound Files

Place your audio files here with the following names:

- `hover.mp3` or `hover.wav` - Sound played when hovering over buttons
- `click.mp3` or `click.wav` - Sound played when clicking buttons

## Supported Formats

The website supports the following audio formats:
- MP3 (.mp3)
- WAV (.wav)
- OGG (.ogg)

## Notes

- Keep sound files small (< 100KB) for fast loading
- Use short sounds (< 500ms) for better UX
- Ensure sounds are normalized to avoid volume inconsistencies
